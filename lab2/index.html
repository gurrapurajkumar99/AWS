Windows Server Deployment on AWS EC2

Step-by-step guide for launching and configuring a Windows Server instance

Overview Requirements Deployment Steps DOS Commands

Overview

This project demonstrates how to launch, configure, and connect to a Windows Server operating system on an AWS EC2 instance. You will also learn how to configure security groups, key pairs, and run basic DOS commands.

Knowledge & Skills Required

  • Basic understanding of AWS services (EC2)
  • Familiarity with DOS command-line interface
  • Basic knowledge of HTML & CSS
  • Networking concepts (IP, ports, security groups)

Deployment Steps

  1. Log in to AWS Management Console.
  2. Navigate to EC2 service and click Launch Instance.
  3. Select Windows Server AMI.
  4. Choose instance type (e.g., t2.micro).
  5. Configure VPC, subnets, and storage.
  6. Create and download a Key Pair.
  7. Configure Security Group (allow RDP port 3389).
  8. Launch instance and wait for it to initialize.
  9. Connect via RDP using Public IP and Administrator password.

Basic DOS Commands

C:\> dir
C:\> ipconfig
C:\> ping google.com
C:\> cls
C:\> exit
